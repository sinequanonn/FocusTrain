# 🚆 FocusTrain

> 기차 여정에 빗댄 집중 서비스

목적지까지 기차를 타듯, 집중하고 싶은 시간을 '여정'으로 설정하고 떠나보세요.
탑승(시작) → 운행(집중) → 하차(일시정지) → 도착(완료)의 흐름으로 집중을 관리합니다.

🔗 **[focus-train.com](https://focus-train.com)**

## 주요 기능

- 출발역·도착역을 골라 나만의 집중 여정 만들기
- 탑승/하차/재탑승으로 집중 세션 관리
- 지난 여정 기록 조회
- Google 계정 간편 로그인

## 🛠 기술 스택

| 영역 | 스택 |
|------|------|
| Frontend | Next.js, Vercel |
| Backend | Java 21, Spring Boot, MySQL |
| Infra | AWS (EC2, ALB, Route53), Docker, GitHub Actions |
| Auth | Firebase Authentication |
| Monitoring | Prometheus, Grafana |
